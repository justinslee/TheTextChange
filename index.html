Welcome to The TextChange

This open source project is aimed at creating a textbook trading online environment that caters to UH ICS students to save money.

Functionality

It has a basic user greeting screen, a navbar to change pages, and an administrator page. The current functionality is mostly within an administrative interface that allows the admin to add, update, edit, or delete students, books, offers, and requests.

Current Development

One item in development is the login authentication using the UH Web Login with the system. A functional PHP code has been developed to authenticate UH students, but the Play Framework doesn't support it. Instead a JavaScript version of the code is still in development. The search function is also being developed so users can see who is offering or requesting books.

Future Development

Future development will include an online version running on CloudBees, substantial error handling, robust application security, and updates to the code and interface.
